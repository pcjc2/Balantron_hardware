# Balantron_hardware

This repository contains board photographs, reverse engineered schematics and debugging notes for the Balantron grinding wheel balancer.

Schematics "\*.sch" are in gEDA (gschem) format.

Also present is an LTspice simulation model which was developed whilst attempting to understand the circuit operation to aid with debugging.

NB: As the schematic was hand created from photographs of the PCBAs, it may very likely contain errors.  Not all components (in particular colour banded capacitors) have had their values confirmed.

Many thanks to RotarySMP (YouTube) for the interesting project.
